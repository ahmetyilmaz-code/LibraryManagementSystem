# Library Management System

C# Windows Forms ve Microsoft SQL Server kullanılarak geliştirilmiş kütüphane yönetim sistemi.

- ## Features
- User Login System
- Add / Update / Delete Books
- Book Lending & Return Operations
- Book Search System
- MSSQL Database Integration

## Teknolojiler (Programs)
- C#
- Windows Forms
- MSSQL

## Kurulum (Setup)
1. SQL scriptini çalıştır
2. Connection string kısmındaki server adını kendi SQL Server instance adınıza göre değiştirin.
3. Visual Studio ile aç
